# 🎓 Student Placement Prediction

A machine learning project that predicts whether a student is likely to be placed based on academic performance, internships, projects, aptitude scores, soft skills, and placement training. Built using Python and Random Forest Classification, achieving **78.15% accuracy**.

## 🚀 Live Demo

https://student-placement-prediction-mfyo3tkdpm5egseix2rvsj.streamlit.app/

## 📊 Dataset Features

* CGPA
* Internships
* Projects
* Workshops/Certifications
* Aptitude Test Score
* Soft Skills Rating
* Extracurricular Activities
* Placement Training
* SSC Marks
* HSC Marks

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Streamlit

## 📈 Model

* Random Forest Classifier
* Accuracy: **78.15%**
